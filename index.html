Welcome to the Pi of Things

Pi of things is a collection of libraries (under development), written in C#, that aim to provide useful APIs to do fun IoT projects using the Raspberry Pi. It is written in C# and uses Mono on the Raspberry Pi.

The GPIO Manager

The GPIOManager wraps the functionality of communicating with the GPIO ports on a Raspberry Pi 2 (and Raspberry Pi 1, B+). It provides useful APIs to communicate with the GPIO using the FileSystem.

Authors and Contributors

The PiOfThings project was originally created by @sumitkm and Raj Paka as a part of their experiments with Raspberry Pi. It is now spun off as a separate library.
